La carpeta system

La carpeta system

La carpeta **system** es donde se encuentra todo el código de CodeIgniter organizado en varias carpetas:

Diferentes subcarpetas que contiene la carpeta system
Nombre Descripción
core Contiene las clases del núcleo de CodeIgniter. Todo nuestro trabajo se realizará en la carpeta application por lo que no tendremos que tocar nada de esta. Si tuviéramos que extender el núcleo de CodeIgniter, en el manual nos recomiendan utilizar hooks (mejoras), el cual disponemos de una carpeta específica para ello.
database Contiene los drivers de bases de datos y otras utilidades. No tendremos que tocar nada de esta carpeta.
fonts La carpeta fuentes contiene información relacionada con fuentes y utilidades. No tendremos que tocar nada de esta.
helpers La carpeta helpers contiene helpers por defecto, tales como date, cookie o URL. En nuestras aplicaciones realizaremos un uso frecuente de estos helpers. En cada versión se suelen añadir o mejorar algunos de los que hay actualmente. Se suelen usar bastante.
language La carpeta language contiene archivos de idioma. Por defecto tendremos una carpeta english/ la cual contiene ficheros PHP específicos utilizados por diferentes helpers, librerías, etc. Cada fichero contiene un array asociativo con mensajes específicos para dicho idioma.
libraries Contiene librerías estándar de CodeIgniter que nos permitirán de forma sencilla enviar emails, gestionar calendarios, subir ficheros, etcétera. Podremos crear nuestras propias librerías o extender (incluso mejorar) las disponibles, pero por defecto deben ser almacenadas en la carpeta application/libraries para mantener separadas las librerías estándar de las librerías creadas por nosotros.
